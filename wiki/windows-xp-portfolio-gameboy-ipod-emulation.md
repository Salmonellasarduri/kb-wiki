# Windows XP風ポートフォリオサイト：動作するGame BoyとiPodを実装した事例

## 概要

ある開発者（mitchivin.com 運営者）が、Windows XPのインターフェースを模したポートフォリオサイトを制作した。このサイトの特徴は、ブラウザ上で実際に動作するGame BoyとiPodのエミュレーションを実装した点にある。

当初はHacker Newsに「Show HN」として投稿されたが、その際は賛否両論の反応を受けた。その後、改良を加えた版を再投稿している。

## 美学としての「廃れたUI」

このプロジェクトに対して寄せられた考察として、「古いUIで包まれた生きたコード」は単なる郷愁ではなく美学の問題として捉えられている。廃れた見た目が新鮮さに転換するという逆説的な現象が興味深い点として挙げられている。
