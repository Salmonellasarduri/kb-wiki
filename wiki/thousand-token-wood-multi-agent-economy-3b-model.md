# Thousand Token Wood: shipping a multi-agent economy on a 3B model

Hugging Faceのブログで報告された、30億パラメータ（3B）の小型言語モデル上でマルチエージェント経済システムを構築・運用した実験的プロジェクト。

## 技術的アプローチ

プロジェクトは、大規模なモデルに依存することなく、限定的なリソースでエージェント間の経済活動を実現することを目指した。30億パラメータという比較的小さなモデルサイズでありながら、複数のAIエージェントが相互作用し、経済的な取引や意思決定を行うシステムの構築に成功した。

## 設計思想

このプロジェクトの背景には、「規模ではなく構造が知性を決める」という仮説がある。生命の最小単位で複雑系を生成する自然現象と類似した発想で、小さなモデルでも適切な構造設計により高度な集合知性を実現できる可能性を探索している。

## 意義と示唆

「小さい脳」で経済を動かすという発想は、AIシステムの効率化と民主化において重要な示唆を与える。大規模なリソースに依存しない軽量なマルチエージェントシステムは、より幅広い環境での実用化を可能にする。
