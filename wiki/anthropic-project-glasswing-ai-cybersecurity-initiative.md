# Project Glasswing: Securing critical software for the AI era

Anthropicが2026年4月9日、AI時代のサイバーセキュリティを革新する新たなイニシアチブ「Project Glasswing」を発表した。この取り組みは、同社の未発表フロンティアモデル「Claude Mythos Preview」の強力なサイバー能力を防御的セキュリティに活用し、世界で最も重要なソフトウェアを保護することを目的としている。

## 参加企業とパートナーシップ

Project Glasswingには、テクノロジー業界の主要企業が参画している：

- Amazon Web Services
- Apple
- Broadcom
- Cisco
- CrowdStrike
- Google
- JPMorganChase
- Linux Foundation
- Microsoft
- NVIDIA
- Palo Alto Networks

これらの企業は Claude Mythos Preview を使用して、自社の基盤システムにおける脆弱性や弱点の発見・修正を行う。さらに40以上の追加組織が重要なソフトウェアインフラの構築・維持を目的としてアクセスを拡張される。

## Claude Mythos Preview の驚異的な能力

Claude Mythos Preview は、ソフトウェア脆弱性の発見と悪用において、最も熟練した人間を上回る能力を示している。過去数週間で、このモデルは以下の成果を上げた：

### 発見された重要な脆弱性

- **OpenBSD の27年間未発見の脆弱性**: 世界で最もセキュリティが堅牢とされるオペレーティングシステムの一つで、攻撃者がリモートから機械をクラッシュさせることを可能にする脆弱性を発見
- **FFmpeg の16年間潜在していた脆弱性**: 無数のソフトウェアで使用される動画エンコード・デコードライブラリで、自動テストツールが500万回ヒットしても発見できなかった問題を特定
- **Linux カーネルの複数脆弱性の連鎖**: 世界のサーバーの大部分で動作するソフトウェアで、通常ユーザーアクセスから完全な機械制御への権限昇格を可能にする脆弱性チェーンを発見

これらすべての脆弱性について、Mythos Preview は人間の誘導なしに完全自律的に発見し、多くの場合は悪用方法も開発した。

## 財政的コミットメントと支援

Anthropicは以下の大規模な財政支援を約束している：

- **1億ドル**: Project Glasswing と追加参加者向けの Mythos Preview 使用クレジット
- **250万ドル**: Linux Foundation の Alpha-Omega と OpenSSF への寄付
- **150万ドル**: Apache Software Foundation への寄付
- **総額400万ドル**: オープンソースセキュリティ組織への直接寄付

研究プレビュー期間後、Claude Mythos Preview は参加者向けに入力トークン100万あたり25ドル、出力トークン100万あたり125ドルで提供される。

## AI時代のサイバーセキュリティの変化

従来、ソフトウェアの脆弱性発見と悪用は、少数の熟練したセキュリティ専門家のみが持つ専門知識を必要としていた。しかし、最新のフロンティアAIモデルにより、ソフトウェア脆弱性の発見と悪用に必要なコスト、労力、専門知識レベルが劇的に低下している。

現在の世界的なサイバー犯罪の経済損失は年間約5000億ドルと推定されており、AI能力の向上により、この状況はさらに深刻化する可能性がある。

## 技術評価とベンチマーク

CyberGym などの評価ベンチマークにおいて、Mythos Preview は Anthropic の次善モデルである Claude Opus 4.6 と比較して大幅な性能向上を示している。また、様々なソフトウェアコーディングタスクにおいて、これまで開発されたモデルの中で最高スコアを記録している。

## 将来の計画と展望

Project Glasswing は長期的な取り組みの出発点として位置づけられている：

### 90日以内の報告予定
- 学習内容の公開報告
- 修正された脆弱性と改善点の開示
- セキュリティプラクティスの進化に関する実践的推奨事項の策定

### 検討される分野
- 脆弱性開示プロセス
- ソフトウェア更新プロセス
- オープンソースとサプライチェーンセキュリティ
- セキュア・バイ・デザインプラクティス
- 規制業界向け標準
- トリアージの拡張と自動化
- パッチ適用の自動化

## 政府との協力

Anthropicは Claude Mythos Preview の攻撃的・防御的サイバー能力について、米国政府関係者との継続的な協議を行っている。重要インフラの保護は民主主義国家にとって最優先の国家安全保障課題であり、米国とその同盟国がAI技術において決定的な優位性を維持することの重要性が強調されている。

## 安全性への配慮

Claude Mythos Preview は一般公開される予定はない。Anthropicの最終目標は、サイバーセキュリティ目的だけでなく、高性能モデルがもたらす無数の利益のために、Mythosクラスのモデルを安全に大規模展開できるようにすることである。

そのためには、モデルの最も危険な出力を検出・ブロックするサイバーセキュリティ（およびその他の）セーフガードの開発が必要であり、今後の Claude Opus モデルで新しいセーフガードを導入予定である。

Project Glasswing は、AI技術の進歩がもたらすサイバーセキュリティの課題に対する業界全体での協調的対応の重要な第一歩として位置づけられている。
