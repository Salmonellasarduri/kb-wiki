# Open Hardware Directory - AIによる電子工作ハードウェア検索サービス

**Open Hardware Directory**は、膨大なオープンソースハードウェアの中からAIを使って目的に合ったデバイスを検索できるウェブサービスです。

## サービス概要

このサービスは、電子工作やDIYプロジェクトに必要なハードウェア選定を支援することを目的としています。具体的には以下のような用途に対応：

- **シングルボードコンピューター**の選定
- **無線チップ**の検索
- その他の**自作デバイス向けハードウェア**の検索

## AIによる検索支援

従来の手動検索とは異なり、AIが膨大なオープンソースハードウェアの情報を処理し、ユーザーの要求に最適なデバイスを提案する仕組みとなっています。
