# OpenAIのAIモデルとCodexがAWSで利用可能に

OpenAIのAIモデルとCodex（コーデックス）ソフトウェア開発AIが、Amazon Web Services（AWS）で利用可能になったことが発表された。

## 企業インフラとの統合促進

この提携により、企業は既存のAWSセキュリティ、請求システム、権限設定をそのまま活用しながら、OpenAIのAIサービスを導入できるようになる。これにより、新たなインフラ構築や複雑な設定変更を行うことなく、AI機能を既存のワークフローに統合することが可能となった。

## 競争地形への影響

記事では、この戦略が「企業インフラの摩擦を取り除くことで普及を加速させる」ものとして位置づけられている。OpenAIとAWSの親密な関係の構築は、競合他社であるAnthropicとの競争環境にも変化をもたらすと分析されている。
