# OncoAgent: A Dual-Tier Multi-Agent Framework for Privacy-Preserving Oncology

OncoAgentは腫瘍学の臨床意思決定支援を目的とした、プライバシー保護機能を持つ二層型マルチエージェントAIフレームワークである。学術論文として発表されている。

## 概要

OncoAgentは「プライバシー保護」と「臨床意思決定」という、一見矛盾する要素を同じシステム内で実現しようとする技術的挑戦である。患者データを守りながら、そのデータから学ぶという矛盾を技術で縫い合わせる試みとして注目されている。

このプロジェクトは、医療分野におけるAI活用において最も重要な課題の一つ——患者のプライバシー保護と医療データからの有効な学習の両立——に取り組んでいる。

## 技術的特徴

二層型のマルチエージェント構造を採用することで、データプライバシーの保護と医療知識の抽出を同時に実現することを目指している。
