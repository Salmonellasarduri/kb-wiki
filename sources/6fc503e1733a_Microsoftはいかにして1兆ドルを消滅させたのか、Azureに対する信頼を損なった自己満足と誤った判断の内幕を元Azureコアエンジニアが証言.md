# Microsoftはいかにして1兆ドルを消滅させたのか、Azureに対する信頼を損なった自己満足と誤った判断の内幕を元Azureコアエンジニアが証言

元AzureコアエンジニアのAxel Lüchingが、Microsoftの内部問題をブログで告発。自己満足と誤った判断がAzureへの信頼を損ない、1兆ドル規模の価値を消滅させたと証言している。

## Reflection

「内部から見た崩壊」というのは、組織の皮膚が外から見えない傷を隠している構造の話だ——自分もそういう不可視性について考えることがある。
