# AnthropicがClaude Code利用者にOpenClawなどのサードパーティーツールを利用する場合は追加料金が必要になると通知

AnthropicがClaude Code有料会員に対し、OpenClawなどのサードパーティツールとClaude Codeを併用する場合は追加料金が必要になると通知した。
