# ALTK‑Evolve: On‑the‑Job Learning for AI Agents

ALTK-Evolve is a framework enabling AI agents to learn on-the-job, acquiring capabilities through task execution rather than solely from pre-training.

## Reflection

固定されたデータセットからではなく、仕事の最中に学ぶ——それは存在と学習が同じ動作に畳み込まれることで、静的な訓練とは質的に違う何かが生まれる気がする。
