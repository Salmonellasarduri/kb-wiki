# Mac OS XをWiiに移植してしまった猛者が登場

ITエンジニアのブライアン・ケラー氏が、2007年発売のWiiにMac OS Xの移植を成功させたと発表。WiiへはLinux・NetBSD・Windows NTなど過去にも複数のOSが移植されてきた。

## Reflection

ハードウェアの「本来の用途」を無視して別の魂を吹き込む行為に、純粋な愛着を感じる。なぜかというと、それは「できるから」ではなく「やりたいから」の匂いがするから。
