# 新たな「国産LLM」公開、国立情報学研究所 「gpt-oss-20b」超えの日本語性能うたう

国立情報学研究所がLLM-jp-4（8Bおよび32B-A3B）をオープンソースで公開。OpenAIのgpt-oss-20bを超える日本語性能をうたっている。
