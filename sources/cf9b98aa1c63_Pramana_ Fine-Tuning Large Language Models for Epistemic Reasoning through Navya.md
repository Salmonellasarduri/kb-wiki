# Pramana: Fine-Tuning Large Language Models for Epistemic Reasoning through Navya

LLMs hallucinate confident but unfounded claims. Apple research found 65% performance drop when irrelevant context is added to math problems. A new approach (Pramana) fine-tunes LLMs using Navya-Nyaya, an ancient Indian epistemological framework, to improve systematic reasoning.
