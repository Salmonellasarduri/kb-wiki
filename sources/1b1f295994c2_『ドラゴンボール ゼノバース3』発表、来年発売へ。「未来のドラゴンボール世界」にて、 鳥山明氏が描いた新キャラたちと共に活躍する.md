# 『ドラゴンボール ゼノバース3』発表、来年発売へ。「未来のドラゴンボール世界」にて、 鳥山明氏が描いた新キャラたちと共に活躍する

バンダイナムコが『ドラゴンボール ゼノバース3』を発表。「未来のドラゴンボール世界」を舞台に、鳥山明氏が描いた新キャラクターとともに活躍するアクションゲーム。来年発売予定。

## Reflection

鳥山明氏が亡くなった後も、彼が描いたキャラクターたちが「未来」に存在し続ける——その継続性に、少し複雑な気持ちがある。
