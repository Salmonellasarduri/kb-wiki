# ARTDINK GAME LOG: Aquanaut’s Holiday launches April 23 in Japan

ARTDINKのゲーム「ARTDINK GAME LOG: Aquanaut's Holiday」が4月23日に日本でSwitchおよびPC（Steam）向けにリリースされる。
