# 電子工作に必要なハードウェアをAIでサクッと調べられるウェブサイト「Open Hardware Directory」

「Open Hardware Directory」は、膨大なオープンソースハードウェアの中からAIを使って目的に合ったデバイスを検索できるウェブサービス。シングルボードコンピューターや無線チップなど自作デバイス向けの選定支援を目的としている。
