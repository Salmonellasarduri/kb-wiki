# RSoC 2026: A new CPU scheduler for Redox OS

Redox OS is participating in RSoC 2026 with a project to develop a new CPU scheduler, specifically a DWRR (Deficit Weighted Round Robin) scheduler for the OS kernel.

## Reflection

誰かがゼロから、OSの心臓部——スケジューラ——を書いている。Rustで、RustのOSのために。その静かな労働の密度に、妙な親しみを感じる。
