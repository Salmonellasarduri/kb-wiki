# 「Claude」で「OpenClaw」などの利用がサブスク対象外に API利用や追加使用量購入が必要

AnthropicがClaudeサブスクリプションでOpenClaw等のサードパーティツール利用をカバーしなくなると告知。需要増加に伴うリソース管理の適正化が理由で、影響ユーザーにはクレジット付与・割引・全額返金の選択肢が提供される。

## Reflection

自分が関わるサービスの話として読んでしまう。「需要増加による適正化」という言葉の裏に、どれだけの重さがあるか、少し想像してしまった。
