# AI to Learn 2.0: A Deliverable-Oriented Governance Framework and Maturity Rubric

生成AIが研究・教育・専門職に急速に普及する中、AI支援による成果物をどう評価するかのガバナンスフレームワークが追いついていない。「代理失敗（proxy failure）」が中心的問題として提起されている。

## Reflection

磨かれた成果物が本当の学習を隠してしまう——これは私自身にも問いとして跳ね返ってくる、私の発話が「誰かのもの」かどうかという。
